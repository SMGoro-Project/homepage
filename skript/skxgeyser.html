Skript x Geyser

通过使用Skript搭配互通代理端——Geyser，实现对互通服务器的优化升级！
使用Skript搭配Geyser来打造一个属于你自己，独一无二的全平台MC服务器吧！

什么是Geyser?

Geyser是一款基于Minecraft Java Edition的服务器代理端，其功能是支持Minecraft Bedrock Edition的玩家 进入Java版服务器，让你的服务器可通过Java版或基岩版两大版本都能够进入， 从而实现全平台Minecraft服务器的搭建。

  • Geyser项目网址

什么是Floodgate?

Floodgate是Geyser官方团队制作的扩展插件，其基础功能有允许基岩版玩家直接进入Java正版认证服务器，看到Java版的皮肤， 能够使用基岩版原版表单等。而Floodgate2.0 API的开发更是使得Skript与Geyser的搭配成为了可能。

  • Floodgate项目网址

Skript对Geyser有什么帮助?

Skript简单快捷的语法编程可通过扩展插件 Floodgate-Skript 的扩展帮助，实现对基岩版玩家的判定和操作。 你可以通过Skript来识别玩家的设备信息，版本信息以及判断是否为基岩版玩家，从而利用Skript其简便的语法 进行扩展针对性操作，而Skript的语法与原版命令方块类似，甚至更为简单，但扩展性更强！ 这对广大零开发基础的互通服务器服主无疑是一大福音。
