什么是Geyser?

Geyser 是一个代理端，允许让Minecraft Bedrock Edition 和 Minecraft Java Edition的玩家在同一个服务器游玩， 该项目的最终目标是让 我的世界基岩版 用户尽可能无缝地加入 我的世界Java版 服务器，以实现真正的跨平台。

如何使用?

Geyser 可以从 Jenkins 服务器下载 点此查看 相关的安装设置可以从我们的 维基页面 或我们的演示视频查看。 Geyser可以用于加入任何Minecraft Java版服务器，而且不需要任何额外操作。

如何工作的?

Geyser 是一个中间代理端，它翻译所有传入和传出的数据包。 话虽如此，Geyser 可以作为独立代理工作，这意味着您可以使用它来加入任何支持最新版的 Minecraft Java 服务器。 Geyser 也可以作为插件安装在您的服务器上，因此您可以轻松地使您的服务器与 我的世界基岩版 兼容。

从任何地方加入

Geyser 可以从 Windows 10、iOS、Android 甚至控制台加入 ( 点击查看 了解如何设置！）间歇泉经过测试，可与 广泛的托管服务提供商, 并且，如果您拥有自己的服务器，您可使用 Floodgate 插件，这将不受 Java 正版帐户的限制！
